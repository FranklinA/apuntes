Diseño y desarrollo web

DISEÑO WEB: Es el diseño visual (look & feel) -> elementos gráficos (tipografías, colores, tamaños y proporciones) Se hace con herramientas gráficas (Photoshop, GIMP) Lo que resulta del diseño NO es código. Es por ejemplo una imagen. Se trocea para hacerlo en HTML + CSS

DESARROLLO WEB: Programación necesaria para construir una aplicación o sitio web. Tiene una parte cliente / parte servidor (funcionan independientes y utilizan diferentes tecnologías aunque interactúan)

  • Cliente: lo que se "ve" ** HTML + CSS y ** JavaScript y DOM
  • Servidor: programación para construir el back-end de un sitio web. Que es la parte de un sitio web que no se ve, pero necesaria para poder presentar la información de forma correcta. ** Se utilizan lenguajes de script como PHP, ASP, ASP.NET, JSP, Perl y Coldfusion) para acceder a BBDD y recuperar información. ** Diseño y desarrollo de BBDD ** Seguridad del producto final

'''Hasta cuando hay que mantener el soporte para un navegador''' Jakob Nielsen y Hoa Loranger "[http://www.agapea.com/libros/Usabilidad-Prioridad-en-el-diseno-Web-9788441520929-i.htm Priorizando la usabilidad web]" sugieren esperar 5-6 años desde el lanzamiento de la nueva versión del navegador. Por ejemplo, en 1999 se lanzó la versión IE5, así que podías ignorar con seguridad la versión IE4 en 2004... y así. Lo mejor es comprobar los registros del servidor para ver qué porcentaje de tus clientes usa esa versión o ese navegador. Cuidado porque el uso de los navegadores también varía según si es día laborable (en el curro) o fin de semana (en casa). Y ahora más con tablets y teléfonos... Incluso Microsoft lanzó una campaña para animar a sus usuarios a dejar IE6...
